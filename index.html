吉他打品咋辦

芝加哥冬天太過乾燥，加上室內空調全天不休，我這兩把吉他相繼出現了打品的問題。總結了這幾個辦法：

  • 吉他音孔裡面靠近上面的地方有一個小洞，找一把尺寸合適的螺絲刀伸進去，轉動螺絲刀，往哪邊轉我忘了。反正就是調節琴頸的高低。轉不動的話，請拿出家裡用的食用油滴兩滴進去。注意別轉的太過了，防止吉他被玩壞。網上說可以目測琴頸的高低，我反正從來都看不出來。
  • 拿一個濕毛巾，擰乾，別太乾。拿一個ziploc的袋子，沒有可以用塑料袋代替，用刀劃幾個口子，把毛巾放進去。袋子口合上，放進吉他琴箱裡72小時。
  • 換一把吉他
  • 其他打磨品絲的操作太高級了，我沒試過

吉他都金貴得很叻，哪怕是最辣雞的合板吉他，也會受濕度的影響。不能太乾燥也不能太潮溼。
南方的小夥伴可以在吉他琴箱裡塞一包乾燥劑，注意更換。北方的小夥伴平時可以在旁邊個一盆水，不用吉他的時候裝回琴包裡。千萬不要放到暖氣旁邊。

set up git on Mac

很久以前，有個朋友告訴我，個人網站對找工作有很大幫助，我沒放在心上。昨天不知道在搜什麼東西，看到了某位大神的blog，造型炫酷，內容紮實，我心動了。
github pages + hexo 當然要從git開始，第一篇文章就想紀錄一下怎麼配置git以及github。廢話有點多。

預備知識

git是啥就不多說了

git有三種主要狀態：committed，modified，和staged

  • committed代表數據已經安全的存入本地數據庫
  • modified代表你對文件進行了更改，而且沒有將更改後的文件committed入數據庫
  • staged代表你已經對更改的文件進行標記進行下一次commit

git project有三個section

如下圖所示：the git directory, the working tree, the staging are

  • git directory是git為你的項目存儲數據的區域，這是git最重要的一部分。從其他計算機克隆倉庫時，拷貝的也是這裡的數據
  • working tree是對項目的某個版本獨立提取出來的內容。也就是你本地的項目，供你修改
  • staging area是一個文件，保存了下次將commit的文件列表信息，一般在git倉庫目錄中

git工作流程：

  • 在working tree（本地）修改文件
  • 暫存文件，將文件快照放入staging area
  • 提交更新。找到暫存區域的文件，將快照永久性存儲到git directory
